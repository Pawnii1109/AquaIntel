## ⚡ Quick Start (Run in this order)

### Step 0 — Create your .env file
```bash
# In D:\mypyprogs\AquaIntel
copy .env.example .env
```
Edit `.env`:
```
DB_HOST=localhost
DB_PORT=3306
DB_NAME=floatchat
DB_USER=root
DB_PASSWORD=YOUR_MYSQL_PASSWORD

GEMINI_API_KEY=YOUR_GEMINI_KEY   ← get free at aistudio.google.com/apikey
GEMINI_MODEL=gemini-1.5-flash
```

---

### Step 1 — Activate your virtual environment
```bash
cd D:\mypyprogs\AquaIntel
myenv\Scripts\activate
```

---

### Step 2 — Create MySQL database and table
First create the database in MySQL Workbench or CLI:
```sql
CREATE DATABASE IF NOT EXISTS floatchat;
```
Then run:
```bash
python scripts/setup_db.py
```
Expected output:
```
✅ Table 'ocean_data' created (or already exists) with all columns.
   Columns: id, latitude, longitude, temperature, salinity, depth, time, float_id
```

```bash
python data_ingestion/db_writer.py
```

## 🗺️ App Features

### 📊 Dashboard Tab
- 5 KPI cards: floats, records, avg temp, avg salinity, max depth
- Live ARGO float position map (colored by temperature)
- Average depth profile (dual-axis: temperature + salinity)
- Monthly temperature trend line chart
- Records per float bar chart

### 💬 Chat Tab
- Type any ocean data question in plain English
- Gemini converts it to MySQL → runs query → generates answer
- Auto-generates the best chart for the answer
- Shows the generated SQL (expandable)
- Download results as CSV

### 🗺️ Explorer Tab
- Map View: colored by temperature or salinity
- Depth Profile: adjustable max depth
- Time Series: monthly temperature or salinity
- T-S Diagram: temperature vs salinity scatter

### 📁 Raw Data Tab
- Filter by depth and temperature
- Browse last 1000 records
- Download filtered data as CSV

## 🔧 Troubleshooting

### ❌ MySQL connection failed
```
sqlalchemy.exc.OperationalError: Can't connect to MySQL server
```
→ Check `DB_HOST`, `DB_PORT`, `DB_USER`, `DB_PASSWORD` in `.env`
→ Make sure MySQL service is running (check Task Manager or Services)
→ Try: `mysql -u root -p` in CMD to verify MySQL works

### ❌ ERDDAP fetch failed / timeout
```
❌ Request timed out. Try a smaller region.
```
→ ERDDAP sometimes has high load. Wait 5 minutes and retry.
→ Try a smaller region in `db_writer.py`:
  ```python
  df = fetch_data(region="default")   # smaller than arabian_sea
  ```

### ❌ Gemini API error
```
google.api_core.exceptions.InvalidArgument: API key not valid
```
→ Check `GEMINI_API_KEY` in `.env`
→ Get free key at: https://aistudio.google.com/apikey
→ Make sure there are no spaces around the key in `.env`

### ❌ FAISS index not found
```
FileNotFoundError: FAISS index not found at vector_store/faiss.index
```
→ Run: `python vector_store/embedder.py`
→ Make sure `ocean_data` table has data first (Step 3)

### ❌ sentence-transformers downloading slowly
→ First run downloads ~90MB model. Let it complete.
→ After that it's cached locally (~/.cache/huggingface/)

### ❌ Streamlit shows blank screen
→ Check terminal for error messages
→ Try: `streamlit run frontend/app.py --server.port 8502`
